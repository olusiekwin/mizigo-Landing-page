## [Mizigo Africa Website](https://mizigoafrica.com/)

This GitHub repository powers [Mizigo Africa](https://mizigoafrica.com/), a logistics platform offering real-time cargo tracking, secure payments, and driver-partner services for seamless cargo transportation across Africa.

### Editing the Site:

Clone this repo and push changes.

The only requirement is having Node.js & npm installed - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

Follow these steps:

```sh
# Step 1: Clone the repository using the project's Git URL.
git clone https://github.com/MizigoAfrica/MizigoAfrica/tree/main/MizigoAfrica_Web

# Step 2: Navigate to the project directory.
cd mizigo_africa

# Step 3: Install the necessary dependencies.
npm i

# Step 4: Start the development server with auto-reloading and an instant preview.
npm run dev
```
